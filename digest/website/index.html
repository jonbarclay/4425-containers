Course Description

In this course, we will examines web application vulnerabilities and remediation techniques. We will explore various tools and techniques used to perform web application assessments. Topics covered in this course include cross-site scripting, SQL injection, session management, and web server configuration. Emphasis is placed on practical skills developed through extensive hands-on exercises.

Course Objectives

Upon successful course completion, students should be able to do the following:
Describe common web application vulnerabilities
Describe web application vulnerability prevention and remediation techniques
Explain assessment methodologies
Demonstrate use of web application assessment tools
Describe how web applications are exploited
Explain the risk and cost to organizations if web applications are exploited
